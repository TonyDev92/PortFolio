Games API

A secure gaming API with path restrictions and security measures. It uses CORS, JSON Web Token, and other security layers. The /games, /platforms, and /user endpoints are available. To test it, clone the repository, install the dependencies with 'npm i' and run 'npm run dev' in the project folder. I recommend using Postman to interact with the API.
